$ 49. Sort Letters by Case

Description

Given a string which contains only letters. Sort it by lower case first and upper case second.

Example
Example 1:
	Input:  "abAcD"
	Output:  "acbAD"

Example 2:
	Input: "ABC"
	Output:  "ABC"
	
Challenge

Do it in one-pass and in-place.	
	
给定一个只包含字母的字符串，按照先小写字母后大写字母的顺序进行排序。
